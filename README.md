# TOT_Springboot_PreAssessmentTest

As a software developer a task is given to you to develop a shopping cart for an online shopping application (you need to develop backend logic in Java programming, not any GUI). In the shopping mall, the following categories belong to products:

Clothes Product
Electronic Product
Toys Product

A user can add any type of product to the cart during shopping and can also perform the following operations:

Add Product
Remove Product
Update Product
Show Cart

(While displaying products from the cart, all added products should be visible in a sorted manner.
